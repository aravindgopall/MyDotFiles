#MyDotFiles

Install oh-my-zsh 
	https://github.com/robbyrussell/oh-my-zsh

then copy the .zshrc contents above to your ~/.zshrc file


Install Powerlevel9K theme
	https://github.com/bhilburn/powerlevel9k

this is the theme i used and ~/.zshrc is configured for this theme... you can download anyother theme and configure the zshrc file



Vim Configurations:
